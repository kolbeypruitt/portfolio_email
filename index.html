King Of The World

Multiplayer Game

View Live   Github  · Full-Stack, Javascript, Websockets, Phaser.Io

This is an open lobby multiplayer game in which each user selects a character and then fights each other. The losers are eliminated, while the those who remain will gain extra health and deal more damage to opponents. The last person alive is considered “king of the world”.


Conway's Game Of Life

Simulation App

View Live   Github  · Javascript, HTML5, CSS

This simulation takes Conway's basic rules a step further by making newborn cells inherit their color based on the taking an R, G, and B color value from each of its 3 neighbors. The newborns also start out with 10% opacity which increases 10% each turn that it lives.
